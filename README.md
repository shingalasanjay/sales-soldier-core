# Sales Soldier - Lead Intelligence Agent

Part of the **RAK AI Studio** suite.

## Purpose
Automate the research and qualification of sales leads for small and medium businesses.

## Initial Capabilities
- [ ] Monitor incoming leads (Email/Webform)
- [ ] Perform deep research on lead company/profile
- [ ] Draft personalized outreach using GPT-4o
- [ ] Human-in-the-loop approval workflow

## Tech Stack
- Python 3.10+
- OpenClaw Agent Integration
- Google Cloud Platform (Backups & Hosting)
